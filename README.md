# SimpleCrud

<h3> This is a basic CRUD ( Create, Read, Update and Delete ) Application made with PHP and MySql </h3>

A user can do the followings in the database: 
<ul> 
  
  <li> Create</li>
  <li> Read</li>
  <li> Update</li>
  <li> Delete</li>
  

</ul>
